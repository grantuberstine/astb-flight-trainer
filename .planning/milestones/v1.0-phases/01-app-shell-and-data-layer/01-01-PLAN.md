       - BaseQuestion interface: id (string), section (enum: MST | RCT | MCT | ANIT | SAT), difficulty (enum: easy | medium | hard), tags (string[]), explanation (string -- required, never optional)
       - MultipleChoiceQuestion extends BaseQuestion: type 'multiple-choice', text (string), options (string[4]), correctAnswer (number 0-3)
       - PassageQuestion extends BaseQuestion: type 'passage', passage (string), questions (array of {text, options, correctAnswer, explanation})
       - SpatialQuestion extends BaseQuestion: type 'spatial', scenarioParams (object with pitch, bank, heading, coastline), correctAnswer (number 0-4)
       - Export a union type Question = MultipleChoiceQuestion | PassageQuestion | SpatialQuestion

       - SectionScore: correct (number), total (number), averageTimeMs (number)
       - SectionId type: 'MST' | 'RCT' | 'MCT' | 'ANIT' | 'SAT'

    7. Create src/types/gamification.ts:
       - Rank enum: CADET, ENSIGN, LIEUTENANT, COMMANDER, PILOT
       - Badge interface: id (string), name (string), description (string), icon (string), earnedAt (number | null)
       - GamificationState: xp (number), rank (Rank), currentStreak (number), longestStreak (number), lastActiveDate (string | null -- ISO date), streakFreezeAvailable (boolean), badges (Badge[])

    8. Create src/lib/constants.ts with:
       - RANK_THRESHOLDS: Record mapping each Rank to its XP threshold (Cadet: 0, Ensign: 500, Lieutenant: 2000, Commander: 5000, Pilot: 10000)
       - XP_VALUES: { correctAnswer: 10, perfectSection: 50, dailyGoal: 25, missionComplete: 100 }
       - ASTB_SECTIONS array with id, name, description, icon name, and color for each section

    9. Update src/main.tsx to call requestPersistentStorage() on startup:
       ```typescript
       async function requestPersistentStorage() {
         if (navigator.storage && navigator.storage.persist) {
           await navigator.storage.persist();
         }
       }
       requestPersistentStorage();
       ```
